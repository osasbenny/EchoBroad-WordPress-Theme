> # EchoBroad Custom WordPress Theme
> 
> This is a fully functional, pixel-perfect custom WordPress theme converted from the EchoBroad redesign project.
> 
> ## Features
> - **100% Visual Identity**: Preserves all original colors, fonts, animations, and layouts.
> - **Responsive Design**: Fully optimized for all devices.
> - **Dynamic Content**: Integrated with WordPress core functions.
> - **Fast Loading**: Optimized asset enqueuing.
> 
> ## Installation
> 1. Download the `echobroad-theme.zip` file.
> 2. Log in to your WordPress Dashboard.
> 3. Go to **Appearance** → **Themes** → **Add New** → **Upload Theme**.
> 4. Choose the ZIP file and click **Install Now**.
> 5. Click **Activate** once the installation is complete.
> 
> ## Theme Structure
> - `style.css`: Theme metadata and basic styles.
> - `functions.php`: Asset enqueuing and theme support configuration.
> - `header.php`: Site header and React entry point.
> - `footer.php`: Site footer and WordPress hooks.
> - `front-page.php`: Main template for the homepage.
> - `assets/`: Compiled CSS and JS from the original project.
> - `images/`: All image assets used in the project.
> - `inc/`: Custom template tags and functions.
